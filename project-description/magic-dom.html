Introducing a lightweight frontend library built on the Vite engine and designed with JSX syntax in mind.

This library brings reactivity and scoped styles to your projects, enhancing interactivity and design consistency.

Creating a JSX element grants you the benefits of a regular DOM element, but with added functionality for smoother development.

The motivation behind this library was to simplify the use of vanilla JavaScript DOM features, often requiring extra steps when working with frameworks like Vue or React.
